Tema 4: Bucles

1. ¿Qué diferencia hay entre bucle while y un bucle for?

El bucle while NO SE EJECUTA en el caso de que la condición sea false.

2. De acuerdo a la sintaxis del bucle while, ¿qué es falso en relación a la condición?

La condición sólo se evalúa al principio de la ejecución del bucle.

3. ¿Qué es una variable centinela?

Es una variable booleana (boolean) a la que asignaremos valores y que nos ayudará en el control del bucle.

4. De acuerdo a la sintaxis del bucle do-while: do{ sentencias } while (condición) Señalar cuál es la afirmación falsa

Si condición = = false, el bucle no se llega a ejecutar nunca.

5. De acuerdo a la sintaxis del bucle for for (inicialización ; condición ; actualización) { sentencias } Señalar cuál es la afirmación falsa:

La inicialización se realiza cada vez antes de entrar al bucle.

6. ¿Cuál de estas características de las sentencia “break” y “continue” es “falsa”?

La sentecias break y continue pueden usarse en tanto en sentencias de repetición (bucles) como de bifurcación (switch).

7. ¿ Cuántas veces se ejecuta el cuerpo del siguiente bucle ? int i = 0; while (true) { i ++; if (i <10) continue; i ++; if (i== 10) break; }

El bucle se ejecuta permanentemente (no sale).
